# WeatherApp
Weatherapp is a simple forecast app, which uses some APIs to fetch 5 days forecast data from the [OpenWeatherMap]
(https://openweathermap.org/forecast5). The main goal of this app is to be a sample of how to build an high quality
 Android application that uses the Architecture components, Retrofit etc. in Kotlin

## Architecture
The app uses MVVM [Model-View-ViewModel] architecture to have a unidirectional flow of data, separation of concern, testability, and a lot more.
